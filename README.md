## Installation

1. Clone the repository:

```https://github.com/yudistirarivaldi/mile-app-test```


2. Install the dependencies:

```composer install```


3. Start the server:

```php artisan serve```

4. Run unit test:

```php artisan test```

5. Documentation API With POSTMAN

```https://documenter.getpostman.com/view/17375603/2s9YR9YCRe```
